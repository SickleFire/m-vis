# Mvis 
**Memory Visualizer Debugger Tool**

Mvis is a powerful debugging utility designed to **scan and visualize process memory allocation**, down to the **heap level**. It helps developers, researchers, and systems programmers gain deep insights into how memory is being used by running processes.

---

##  Features
-  **Process Scanning**: Inspect memory allocations of active processes.
-  **Heap-Level Analysis**: Dive into heap structures for detailed debugging.
